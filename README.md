# Nexcan Digital Service

Digital Growth Agency website built with modern web technologies.

## Technology Stack

- **Vite** for fast development and building
- **React** for the user interface
- **TypeScript** for type-safe code
- **Tailwind CSS** for modern and responsive styling
- **Shadcn UI** for high-quality reusable components
- **Framer Motion** for smooth and premium animations

## Getting Started

Follow these steps to run the project locally:

1. **Clone the repository**:
   ```sh
   git clone <repository-url>
   ```

2. **Navigate to the project directory**:
   ```sh
   cd nexcan---DigitalMarketing
   ```

3. **Install dependencies**:
   ```sh
   npm install
   ```

4. **Start the development server**:
   ```sh
   npm run dev
   ```

The application will be available at `http://localhost:8080` (or the port specified in the terminal).

## Deployment

The project can be built and deployed as a static website.

To build the project for production:
```sh
npm run build
```

The output will be generated in the `dist` folder.
